hexo使用emoji

2017-05-25
随笔

哪怕你的问题只涉及一点点技术问题，都请务必遵守“文明上网”。
我想着在hexo里用emoji表情，度娘找到的基本都是垃圾，又费尽周折地看了一堆废话。

步骤

注意：所有的操作都是在hexo目录下完成的，例如H:\Hexo

1.下载 emojy
npm install https://github.com/frapples/hexo-article-emoji --save

2.安装 emojy
hexo article-emoji install full

3.配置
将hexo的 _config.yml找到post_asset_folder改为true
post_asset_folder: true
并添加下面语句
article_emoji:
full: true

4.使用
例如笑哭2333: joy:
我这里:和j之间多了一个空格，为的是展现用法，实际用的话，去掉这个空格就可以了。
在markdown编辑器里可能无法显示效果但上传之后会显示的。
